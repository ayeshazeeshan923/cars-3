# 🏎️ Disney Pixar Cars 3 — Browser Racing Game

A fan-made browser racing game featuring all 27 characters from Disney Pixar's **Cars 3**. Play directly in your browser — no installation needed!

🔗 **Live Game:** [ayeshazeeshan923.github.io/cars3-game](https://ayeshazeeshan923.github.io/cars3-game/)

---

## 🎮 How to Play

### Desktop (PC / Laptop)
| Key | Action |
|-----|--------|
| `W` or `↑` | Accelerate / Gas |
| `S` or `↓` | Brake |
| `A` or `←` | Steer Left |
| `D` or `→` | Steer Right |

### Mobile / Tablet
On-screen buttons appear automatically:
**◀ LEFT** &nbsp;|&nbsp; **⛽ GAS** &nbsp;|&nbsp; **🛑 BRAKE** &nbsp;|&nbsp; **RIGHT ▶**

## ✨ Game Features

- **27 playable Cars 3 characters** — each with unique Speed, Handling & Boost stats
- **Car Selection Screen** — pick your favorite racer before the race
- **3-Lap Championship Race** against 5 AI opponents
- **⚡ Boost Pickups** — collect lightning bolts for a speed burst
- **🟣 Oil Slicks** — avoid purple puddles or you'll spin out!
- **Live HUD** — Speed, Lap counter, Position, Race timer
- **Mini-map** — track your position vs opponents in real-time
- **Sound Effects** — engine revving, boost, crash, countdown, lap complete, win/lose fanfare
- **Fully Responsive** — works on desktop, tablet, and mobile
- **No installation** — single HTML file, runs in any browser
